Cinema 4D and BodyPaint 3D make use of a number of open source libraries.  Each one have 
different compliance requirements.

This directory contains archives of open source libraries which require to be 
redistributed. Modified open source archives contain changes that were required by
MAXON.

For more information about open source libraries used in Cinema, see the Copyrights
section of the Cinema 4D / BodyPaint 3D documentation.
